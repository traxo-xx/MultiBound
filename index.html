What is Multibound?

Did you not read the tagline?

"A custom implementation of the Starbound server."

It is in a very untidy and broken mess right now, so if you want something that works, I suggest using StarryPy. For a long while.

How can I help?

Go read the table below and create a pull request with code that implements that specific packet.

If you don't want to implement a packet, just create a pull request anyway.
